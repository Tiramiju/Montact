# 📇montact
>비즈니스 연락처 관리 앱

개발환경 : Android sdk 26+   
개발언어 : Java   
## 사용 라이브러리
 - Lottie : 프로세싱 애니메이션 뷰   
 - material-calendarview : 캘린더 뷰   
 - tess-two : Tesseract ORC 스캐너   
 - Room : 로컬 데이터베이스 구축
